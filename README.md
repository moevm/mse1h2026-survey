# mse-template

## Установка и запуск
#### 1. Клонировать репозиторий

```bash
git clone https://github.com/moevm/mse1h2026-survey.git
```

#### 2. Перейти в каталог проекта

```bash
cd mse1h2026-survey
```

#### 3. Запустить контейнеры

##### Linux

```bash
sudo docker compose up --build
```

##### Windows

```bash
docker compose up --build
```

#### 4. Применить миграции базы данных

После запуска контейнеров открой новый терминал и выполни:

##### Linux

```bash
sudo docker compose exec backend alembic upgrade head
```

##### Windows

```bash
docker compose exec backend alembic upgrade head
```

Команду необходимо выполнять после полного запуска контейнеров.

#### 5. Открыть приложение

Frontend:
http://localhost:8080

Backend API:
http://localhost:8080/api

Swagger:
http://localhost:8080/api/docs

#### Остановка контейнеров

```bash
docker compose down
```
## Проверка работоспособности

Ниже приведён порядок проверки основного пользовательского сценария и результатов работы системы.

### 1. Проверить статус всех модулей приложения

Открыть страницу:

```text
http://localhost:8080/health
```
Убедиться, что frontend, backend и база данных находятся в рабочем состоянии.

#### 2. Начать прохождение опроса:

```text
http://localhost:8080
```
Ввести номер группы в соответствующее поле и нажать кнопку «Начать опрос».

#### 3. Ответить на вопросы

Последовательно ответить на все вопросы, представленные на странице опроса.

#### 4. Завершить опрос

Нажать кнопку «Закончить опрос».

После этого система должна:

- отправить ответы на backend
- сохранить результаты в базе данных
- отобразить страницу успешного завершения опроса

#### 5. Проверить сохранение ответов

Для проверки сохранённых данных открыть Swagger:

```text
http://localhost:8080/api/docs
```

Авторизоваться с учётными данными:

```text
login: admin  
password: admin  
```

После авторизации выполнить запрос к endpoint:
```text
GET /survey/answers/{id}
```
В качестве параметра id указать:
```text
1
```
Убедиться, что в ответе отображаются сохранённые ответы на опрос с id = 1.
