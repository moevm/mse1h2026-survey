# mse-template

## О проекте

### Постановка задачи

Система кафедральных опросов — это веб-приложение для проведения опросов, хранения результатов и анализа полученных данных.

Основные цели системы:

- проведение опросов через веб-интерфейс
- хранение результатов опросов в базе данных
- просмотр результатов опросов
- анализ данных
- визуализация статистики
- экспорт результатов

Система предоставляет:

- пользовательский интерфейс для прохождения опросов
- административную панель для управления опросами
- REST API для взаимодействия frontend и backend
- хранение данных в PostgreSQL

Проект является развитием существующей системы:

https://survey.moevm.info/

---

### Требования

Основные функциональные требования:

Система должна обеспечивать:

- создание опросов
- редактирование опросов
- удаление опросов
- прохождение опросов по уникальной ссылке
- хранение ответов
- просмотр статистики
- экспорт данных
- визуализацию данных

Типы вопросов:

- **free** — свободный ответ
- **checkbox** — множественный выбор
- **radiobutton** — один вариант ответа
- **scale** — оценка от 1..N

Технологии:

**Backend**

- FastAPI
- SQLAlchemy
- PostgreSQL
- Pydantic

**Frontend**

- React
- JavaScript

**Infrastructure**

- Docker
- nginx

Нефункциональные требования:

- поддержка не менее 100 одновременных пользователей
- время ответа API не более 5 секунд
- безопасная авторизация пользователей
- хранение паролей в зашифрованном виде
- модульная архитектура

---

### Сценарии использования

#### Прохождение опроса

1. Пользователь открывает ссылку вида:

`/survey/{survey_id}`

2. Система проверяет статус опроса.

3. Если опрос активен:

- отображается название
- отображается описание
- отображаются вопросы

4. Пользователь отвечает на вопросы.

5. Пользователь отправляет ответы.

6. Система сохраняет ответы в базе данных.

7. Отображается сообщение об успешной отправке.

---

#### Управление опросами (Администратор)

1. Администратор входит в систему.

2. Администратор открывает административную панель.

3. Администратор может:

- создать опрос
- редактировать опрос
- удалить опрос
- включить опрос
- отключить опрос

---

#### Просмотр статистики

1. Администратор открывает страницу статистики.

2. Система отображает:

- количество ответов
- список вопросов
- распределение ответов

3. Система строит графики.

---

### Инструкция по запуску

#### 1. Клонировать репозиторий

```bash
git clone https://github.com/moevm/mse1h2026-survey.git
```

#### 2. Перейти в каталог проекта

```bash
cd mse1h2026-survey
```

#### 3. Запустить контейнеры

##### Linux

```bash
sudo docker compose up --build
```

##### Windows

```bash
docker compose up --build
```

#### 4. Применить миграции базы данных

После запуска контейнеров открой новый терминал и выполни:

##### Linux

```bash
sudo docker compose exec backend alembic upgrade head
```

##### Windows

```bash
docker compose exec backend alembic upgrade head
```

Команду необходимо выполнять после полного запуска контейнеров.

#### 5. Открыть приложение

Frontend:
http://localhost:8080

Backend API:
http://localhost:8080/api

Swagger:
http://localhost:8080/api/docs

#### Остановка контейнеров

```bash
docker compose down
```
